NASM
=====

`SASM - simple crossplatform IDE for NASM, MASM, GAS and FASM assembly languages
<https://github.com/Dman95/SASM>`_

::

        # [format] can be       elf32, elf64 (Linux)
        #                       win32, win64 (Windows)
        nasm -f [format] file.asm
